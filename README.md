# Assignment 3 – Supervisor Pattern Use Case

This assignment explores the **Supervisor Pattern** in multi-agent systems.

### Use Case
Customer Support, Billing, and Technical Troubleshooting Assistant.

The system uses a **Supervisor Agent** to coordinate multiple specialized sub-agents:

- Technical Support Agent
- Billing Agent
- Communication Agent

The supervisor analyzes the user request and routes tasks to the appropriate agent, combining the results into a single response.

This architecture demonstrates why complex workflows often require **multiple specialized agents instead of a single agent**.
